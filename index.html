⚔️ Jedi Balance & Expertise Update

Major improvements to Light Side support, Dark Side damage scaling, core Jedi progression, and Heroic Jewelry sets.

🛡️ Light Side Support

  • Fixed Soothing Aura threshold values so Light path healing buffs apply correctly.
  • Increased Soothing Aura healing bonuses, allowing full Light path Force Regeneration to heal for approximately 15K.
  • Added a controlled Forsake Fear group health HoT while preserving the existing action‑heal channel behavior.
  • Reworked the unused Lethargy line into Improved Forsake Fear functionality.
  • Improved Forsake Fear now reduces cooldown, increases HoT strength, and expands group radius.
  • Preserved existing behavior for Cloud Minds, Expansive Trickery, and Mind Trick.

🌑 Dark Side Damage Scaling

  • Fixed Dark Side threshold‑gated expertise values for Tempt Hatred and Wracking Energy.
  • Added direct Dark Side Force power damage scaling through active Jedi path expertise.
  • Increased scaling for Dark Lightning, Maelstrom, Force Drain, and Improved Force Choke.
  • Avoided shared NPC combat data changes – these buffs only affect intended player Jedi paths.

✨ Core Jedi Progression

  • Increased Force Sensitive core stat progression from levels 1–90.
  • Added baseline Jedi innate protections to phase master rows.
  • Increased selected Jedi expertise bonuses for damage, block, armor bypass, and healing.
  • Regenerated active Jedi skill and expertise files from clean source to clear stale FRS test mismatches.

💍 Heroic Jewelry Update

Guardian Set

  • Buffed Guardian heroic jewelry defensive stats.
  • Reworked the 3‑piece, 4‑piece, and 5‑piece Guardian bonuses within the five‑effect buff limit.
  • Increased Guardian full‑set heroic token cost to 120 tokens.

Lightsaber Duelist Set

  • Buffed Lightsaber Duelist per‑piece stats.
  • Added true lightsaber weapon damage mods to the Saber DPS set.
  • Reworked the 3‑piece, 4‑piece, and 5‑piece Duelist bonuses within the five‑effect buff limit.
  • Increased Lightsaber Duelist full‑set heroic token cost to 120 tokens.
