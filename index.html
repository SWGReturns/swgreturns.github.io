Launcher Setup Guide

Follow these steps to install and connect.

1. Install SWG

  • Install a clean Star Wars Galaxies client
  • Do not run the original SOE launcher

2. Install Launcher

  • Extract into your SWG directory
  • Example: C:\StarWarsGalaxies\

3. Patch

  • Run the launcher
  • Allow full patching to complete

4. Create Account

  • Create account via launcher
  • No email verification required

5. Play

  • Click Play
  • Log in and enjoy
